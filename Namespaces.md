**Purpose**: Organizes classes to avoid name conflicts.
- **Syntax**:
  ```csharp
  namespace namespace_name
  {
      // define classes here
  }
  ```
- **Example**:
  ```csharp
  namespace School
  {
      class Student { }
      class Course { }
  }
  ```
- **Using Namespaces**: Import with `using` keyword at the top of the file.
